# Fenrok Online - Downloads

Terminal-based MMO inspired by Ragnarok Online and Tibia.

## Downloads

Check the [Releases](https://github.com/Tech-Byte-Frontier/fenrok-releases/releases) page for the latest version.

### Game Client

Play the game in your terminal.

| Platform | File |
|----------|------|
| Linux | `fenrok-vX.X.X-linux` |
| Windows | `fenrok-vX.X.X-windows.exe` |

### Map Editor

Create and edit game maps (for content creators).

| Platform | File |
|----------|------|
| Linux | `fenrok-editor-vX.X.X-linux` |
| Windows | `fenrok-editor-vX.X.X-windows.exe` |

## Quick Start

### Client
1. Download the client for your OS
2. **Windows**: Just run the `.exe`
3. **Linux**: Run `chmod +x fenrok-*-linux` then `./fenrok-*-linux`

The game connects automatically to the server - no configuration needed!

### Editor
1. Download the editor for your OS
2. Run the editor
3. Create/edit maps and save as `.toml` files
4. Maps can be loaded into the game server
