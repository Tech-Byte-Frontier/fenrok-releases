# Fenrok Online - Downloads

Terminal-based MMO inspired by Ragnarok Online and Tibia.

## Download

Check the [Releases](https://github.com/Tech-Byte-Frontier/fenrok-releases/releases) page for the latest version.

### Quick Start

1. Download the appropriate file for your OS
2. **Windows**: Just run the `.exe`
3. **Linux**: Run `chmod +x fenrok-*-linux` then `./fenrok-*-linux`

The game connects automatically to the server - no configuration needed!
